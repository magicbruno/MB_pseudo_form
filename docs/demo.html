What is it

Interceptor lets you control the behavior of any html element when entering or exiting from the viewport.

It uses IntersectionObserver API that is supported by all modern browsers

See on caniuse.com

Interceptor in a nutshell

Ease of use Very easy

Resource consuming Very low

Weight Tiny (2Kb)

Browsers support 96%

const observer = new Interceptor([
  { name: 'animate', action: 'in-viewport' },
  {
    name: 'skill', action: (target, ratio) => {
      let percent = target.dataset.percent;
      if (ratio === 0) {
        target.style.setProperty('--bar-width', '0');
      } else if (ratio === 1) {
        target.style.setProperty('--bar-width', percent);
      }
    }
  },
  {
    name: 'section', action: (target, ratio) => {
      if (ratio >= 0.9) {
        const nextElement = target.nextElementSibling;
        const navButton = document.querySelector('#btn-navpage');
        let id = null;
        if (nextElement) id = nextElement.id;
        if (id) {
          navButton.href = `#${id}`;
          navButton.querySelector('svg').classList.remove('rotate-180');
          navButton.querySelector('span').innerText = 'Next';
        } else {
          navButton.href = `#`;
          navButton.querySelector('svg').classList.add('rotate-180');
          navButton.querySelector('span').innerText = 'Top';
        }
      }
    }
  }
], [0,0.9]);
